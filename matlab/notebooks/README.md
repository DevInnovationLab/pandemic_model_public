# MATLAB Notebooks

Exploratory development notebooks (`.mlx` files). These are not part of the
model workflow and are not guaranteed to run without manual setup.

Each notebook is numbered and explores a specific sub-problem (e.g., surveillance
costs, capacity dynamics, exceedance curves). They were used during model development
and are retained as a scientific record.

To open: launch MATLAB, open the project (`run('./matlab/load_project')`), then
open any `.mlx` file from the MATLAB editor.
